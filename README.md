Search_text
================

Инструмент для поиска текста в книгах и отправки результатов на почту, которые включают в себя название книги, главу и страницу.

Установка
----------

```sh
$ mkdir search_text
$ cd search_text
$ git clone https://github.com/PielliaVasyl/serch_text.git
$ cd serch_text
$ virtualenv .
$ Scripts\activate
$ pip install -r requirements.txt
$ python manage.py runserver
```

Готово

Изменить БД книг
-----------------------------------------

Если Вы хотите изменить Базу данных книг, то неопходимо изменить файл fill_in_db.py по готовому примеру.
После этого в терминале запустить Python и скопировать туда текст из fill_in_db.py.
Осторожно! После этого БД книг и индекс перезапишутся.



Как использовать?
-----------------

```sh
$ python manage.py runserver
```
В браузере открыть http://127.0.0.1:8000/.
Заполнить тескт для поиска, почту, куда отправить письмо с результатами, и лимит времени поиска.
Нажать кнопку Search.
Проверить свою почту на наличие письма с результатами.

Валидация
-----------------
Текст для поиска до 150 символов.
Верный адрес почты.
Лимит времени поиска положительное целое количество секунд. (>=1)
